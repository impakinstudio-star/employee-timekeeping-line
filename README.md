# ระบบเวลาพนักงาน

เว็บแอปต้นแบบสำหรับลงเวลาพนักงานผ่าน LINE, จัดตารางเวร, สรุปวันทำงาน และรวมค่าแรงรายวัน

## เปิดระบบ

ใช้ Node.js เปิดไฟล์ `server.js`

```powershell
$env:PORT='8132'
node server.js
```

จากนั้นเปิด:

```text
http://127.0.0.1:8132/
```

## เข้าสู่ระบบผู้จัดการ

ค่าเริ่มต้นสำหรับทดสอบ:

```text
ผู้ใช้: manager
รหัสผ่าน: 1234
```

ตอนใช้งานจริงให้ตั้งค่าด้วย environment variable:

```powershell
$env:MANAGER_USER='manager'
$env:MANAGER_PASSWORD='your-strong-password'
```

API สำหรับแก้ไขข้อมูล เช่น พนักงาน เวร แคมเปญ และค่าแรง ต้องผ่าน login ผู้จัดการก่อน

## ข้อมูลที่ระบบเก็บ

ข้อมูลอยู่ในไฟล์:

```text
data.json
```

ระบบจะสร้างไฟล์นี้อัตโนมัติเมื่อเปิดใช้งานครั้งแรก

## API สำคัญ

```text
GET  /api/state
POST /api/reset
POST /api/employees
PATCH /api/employees/:id
POST /api/campaigns
DELETE /api/campaigns/:id
PATCH /api/shifts/:id
POST /api/schedule/auto
POST /api/line/command
POST /api/line/bind
GET  /api/payroll?month=2026-07
POST /webhook/line
```

## LINE webhook

ตั้งค่า Webhook URL ของ LINE Official Account ให้ชี้มาที่:

```text
https://your-domain.com/webhook/line
```

ตอนเปิดใช้งานจริงให้ตั้งค่า environment variable:

```powershell
$env:LINE_CHANNEL_SECRET='your-line-channel-secret'
```

ถ้าไม่ได้ตั้งค่า `LINE_CHANNEL_SECRET` ระบบจะรับ webhook โดยไม่ตรวจ signature เพื่อให้ทดสอบในเครื่องได้ง่าย

## คำสั่ง LINE ที่รองรับ

```text
เข้างาน
ออกงาน
ลา วันนี้
หยุด วันนี้
สรุปของฉัน
```

## กะทำงาน

```text
กะเช้า 07:00-15:00
กะบ่าย 15:00-23:00
กะดึกพิเศษ 23:00-07:00
```

กะดึกพิเศษจะเปิดเฉพาะวันสิ้นเดือนหรือวันที่ตั้งเป็นแคมเปญ

## ฟีเจอร์ล่าสุด

- หน้าพนักงานแก้ชื่อและ LINE user ID ได้
- ลบพนักงานได้จากหน้าพนักงาน
- สรุปค่าแรงเป็นรายอาทิตย์ โดยนับช่วงจันทร์ถึงอาทิตย์ตามวันที่ที่เลือก
- ตารางเวรแสดงเป็นปฏิทินรายสัปดาห์
- ลากชื่อพนักงานไปวางในกะที่ต้องการได้
- เอาพนักงานออกจากเวรได้ด้วยปุ่ม × บนชื่อ หรือลากชื่อไปวางในช่องเอาออกจากเวร
- ปฏิทินแสดงชื่อวันจันทร์ถึงอาทิตย์
- หน้าพนักงานตั้งวันหยุดประจำรายคนได้
- หน้าพนักงานตั้งกะที่ทำได้รายคนได้
- หน้าพนักงานใช้ปุ่มบันทึกทั้งหมดปุ่มเดียวด้านล่างตาราง
- ระบบจัดเวรและลากวางจะกันไม่ให้ลงวันหยุดประจำ
- ระบบกันการควงกะในวันเดียวกัน
- ระบบกันกรณีเข้ากะบ่ายแล้ววันถัดไปมาเข้ากะเช้า
